Privacy Policy

1. Purpose of Data Collection and Use

  • EmoPilot stores each user's CloudKit ID on its server for user authentication and usage tracking.
  • The CloudKit ID is saved during the free trial period and remains the same after a subscription begins.
  • This information is retained for one year even if the user does not subscribe or cancels their subscription.

2. Data Retention and Abuse Prevention

  • To prevent repeated abuse of the free trial, EmoPilot retains CloudKit IDs for up to one year.
  • This data is used solely to determine usage restrictions and will not be used for any other purpose or shared externally.

3. When Using the "Advice" Feature

  • When using the "Advice" feature, the selected emotional state and any input are transmitted to the AI server.
  • This data is used only to generate advice and is not stored on either the EmoPilot or AI servers, nor used for any other purpose.

4. Data Security Measures

  • All records are stored in the user’s personal iCloud and follow Apple’s security protocols.
  • EmoPilot uses encryption and access control measures to protect user data.

5. Cookies and Logs

  • This website may collect visit logs (IP, browser info, etc.) in server log format, used solely for performance analysis.

6. No Third-Party Sharing

  • EmoPilot does not sell or share user information with third parties. Data is used only to provide app features and for statistical analysis.

7. User Rights

  • EmoPilot does not accept direct requests for data deletion, but enforces an automatic deletion policy after a set period.

8. Consent

  • By tapping “Agree” and selecting “Free Trial” or “Subscribe,” you are considered to have agreed to this privacy policy.

9. Policy Changes

  • EmoPilot may revise this policy in accordance with legal or service changes and will notify users of such changes on this page.
